# 💎 SkyShop - Free Fire Auto Diamond Topup Laravel Script (Open Source)

**SkyShop** একটি আধুনিক ও সম্পূর্ণ অটোমেটেড ডায়মন্ড টপ-আপ ওয়েবসাইট স্ক্রিপ্ট যা লারাভেল (Laravel) ফ্রেমওয়ার্ক দিয়ে তৈরি। এটি মূলত গেমারদের জন্য একটি ওয়ান-স্টপ সলিউশন। এই স্ক্রিপ্টটি ওপেন সোর্স হিসেবে সবার জন্য উন্মুক্ত করা হয়েছে।

## 🚀 মূল ফিচারসমূহ (Key Features)
- **Auto Player Name Search:** প্লেয়ার ইউআইডি (UID) দিলেই সিস্টেম অটোমেটিক গেমের নাম খুঁজে বের করবে।
- **Auto Add Money:** বিকাশ, নগদ ও রকেটের মাধ্যমে অটোমেটিক ব্যালেন্স অ্যাড করার সুবিধা।
- **Full Admin Control:** শক্তিশালী এডমিন প্যানেল যেখান থেকে প্রোডাক্টের দাম, ইউজার এবং অর্ডার কন্ট্রোল করা যায়।
- **Telegram Notification:** প্রতিটি অর্ডার বা পেমেন্টের আপডেট সরাসরি আপনার টেলিগ্রামে চলে আসবে।
- **Modern UI/UX:** অত্যন্ত ক্লিন, ফাস্ট এবং মোবাইল ফ্রেন্ডলি ডিজাইন।
- **Secure System:** লারাভেল ফ্রেমওয়ার্ক হওয়ার কারণে এটি অত্যন্ত সুরক্ষিত এবং বাগ-ফ্রি।

## ⚙️ সেটআপ গাইড (Setup Instructions)

আপনার হোস্টিংয়ে ওয়েবসাইটটি লাইভ করতে নিচের ধাপগুলো অনুসরণ করুন:

1. **ফাইল আপলোড:** আপনার cPanel-এর `public_html` ফোল্ডারে ডাউনলোড করা জিপ ফাইলটি আপলোড করুন এবং এক্সট্রাক্ট (Extract) করুন।
2. **ডাটাবেজ তৈরি:** cPanel থেকে একটি নতুন MySQL Database তৈরি করুন এবং ইউজার নেম ও পাসওয়ার্ড সেট করুন।
3. **SQL ইমপোর্ট:** `phpMyAdmin`-এ গিয়ে আপনার তৈরি করা ডাটাবেজে রিপোজিটরির সাথে থাকা ডাটাবেজ ফাইলটি ইমপোর্ট করুন।
4. **.env কনফিগারেশন:** আপনার মেইন ডিরেক্টরিতে থাকা `.env` ফাইলটি এডিট করুন। সেখানে আপনার `DB_DATABASE`, `DB_USERNAME`, এবং `DB_PASSWORD` আপডেট করুন। এছাড়াও `APP_URL`-এ আপনার ডোমেইন লিঙ্কটি দিন।
5. **পারমিশন:** নিশ্চিত করুন যে `storage` এবং `bootstrap/cache` ফোল্ডারগুলোতে ফোল্ডার পারমিশন দেওয়া আছে।
